2. **Test build:** `npm run build`
3. **Push to GitHub**
4. **Create Neon database** (5 min)
5. **Deploy to Vercel** (5 min)
6. **Enable Blob storage** (2 min)
7. **Push schema:** `npx prisma db push`
8. **Redeploy**
9. **Test live site!**
